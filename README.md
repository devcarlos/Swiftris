# Swiftris

Tetris Game written in Swift and SpriteKit.

## Features

* Swift 2+
* Support `SpriteKit`
* Easy Game Play and Customization

## Install (iOS 8+)

### GitHub

Clone from GitHub

```
git clone https://github.com/devcarlos/Swiftris.git
```
